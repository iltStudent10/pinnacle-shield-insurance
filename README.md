# Pinnacle Shield Insurance

## Project Description
Pinnacle Shield Insurance is a professional, responsive website for an insurance company. The site provides customers with information about insurance services, allows them to request quotes, and answers frequently asked questions. This full-stack project demonstrates modern web development practices with a focus on user experience and accessibility.

## Live Site
[View the live site on Github Pages](https://iltstudent10.github.io/pinnacle-shield-insurance/)

## Features Implemented
- **Home Page** - Hero section with company introduction and call-to-action
- **Get a Quote Page** - Interactive quote request form with dynamic quote generation
- **About Us Page*** - Detailed company information and mission statement
- **FAQ Section** - Frequently asked questions with interactive accordion functionality
- **Responsive Navigation** - Mobile-friendly navbar with collapsible menu
- **Responsive Deign** - Mobile-first layout that works seamlessly across all devices
- **Bootstrap Integration** - Professional styling using Bootstrap 5.3.2
- **Branding** - Branded color scheme with blue gradient hero section
- **Interactive JavaScript** - Dynamic functionality for quotes, FAQs, and form handling

## Technologies Used
- **HTML5** - Semantic markup for accessibility and SEO
- **CSS3** - Custom styling with CSS variables for consistent branding
- **JavaScript** - Interactive features and form handling (main.js, quote.js, faq.js)
- **Bootstrap 5.3.2** - Responsive CSS framework and components
- **Flexbox & Grid** - Modern CSS layout techniques for responsive design
